# QR-IMG-generator





## Deployment

To deploy this project run

1. Go to the respected Directory

```bash
  git clone https://github.com/chandupriya1206/QR-IMG-generator
```
2. Once its Cloned
```bash
  npm install
```

```bash
  node app.js
```

3. Enter your URL
4. There you go with a QR image in png format
